## Install

```bash
npm install @sinchang/12306cli -g
```

## Demo

[![asciicast](https://asciinema.org/a/157068.png)](https://asciinema.org/a/157068)

## Usage

```js
直接输入命令 12306cli 即可

需要进行间隔刷新，加上 option `12306cli --refresh`，默认 5s，可以自行添加毫秒数
```
